== Description ==
Hide Categories Or Products On Shop Page is woocommerce extension.Hide products and category from shop page woocommerce.This extension is free and it's used with woocommerce.

## Features

- Hide categories in shop page
- Hide products in shop page
- Hide products and categories in shop page
- Select multiple categories
- Hide categories in Widget
- Hide categories/products and Show The 404 Page
